# MarkPad 更新日志

## v0.2.0 - 2024-11-04

### 🎉 新增功能

#### 数据持久化
- ✅ **自动保存**: 数据自动保存到本地 JSON 文件
- ✅ **启动加载**: 应用启动时自动加载历史数据
- ✅ **数据位置**: 数据存储在系统应用数据目录 `AppData/markpad/markpad-data.json`

#### 项目/类型/标签管理
- ✅ **项目管理对话框**: 
  - 添加/编辑/删除项目
  - 自定义项目颜色（8种颜色可选）
  - 实时预览
- ✅ **类型管理对话框**:
  - 添加/编辑/删除便签类型
  - 自定义图标（5种图标可选）
  - 待办、笔记、想法、会议等类型
- ✅ **标签管理对话框**:
  - 添加/编辑/删除标签
  - 标签自动关联到便签

#### 数据管理
- ✅ **导出数据**: 将所有数据导出为 JSON 文件
- ✅ **导入数据**: 从 JSON 文件导入数据
- ✅ **备份数据**: 创建带时间戳的备份文件

### 🎨 UI 优化
- ✅ 侧边栏添加快捷管理按钮（+ 号）
- ✅ 顶部栏添加数据管理菜单
- ✅ Modal 对话框组件（支持 ESC 键关闭）
- ✅ 颜色选择器
- ✅ 图标选择器

### 🔧 技术改进
- ✅ Tauri 命令实现文件读写
- ✅ React Hook 实现数据持久化
- ✅ Zustand Store 统一状态管理
- ✅ 防抖优化自动保存（1秒延迟）

### 📝 已知问题
- ⚠️ 需要安装 Rust 环境才能运行 Tauri
- ⚠️ 数据导入会覆盖现有数据（无合并功能）
